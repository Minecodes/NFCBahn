# Tools

## Mobile Apps

 - **NFC Tools** (Android / iOS): Used for reading early reading and on occasional reads, when other tools are not available.

## Hardware
  - **Flipper Zero**: Current tool for reading and collecting the NFC data. There was no encryption or protection detected or cracked. All readings were done using the built-in NFC module, and no modifications were necessary.
